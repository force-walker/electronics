# MVP仕様書 v0.1

## Target
- 中〜高校生
- 見た目: 教材寄り（低負荷）
- 言語: 日本語 / 英語

## Stages
1. 電圧・電流・抵抗（直感）
2. オームの法則（計算）
3. 直列・並列（配線）

## Metrics
- Voltage (V)
- Current (A)
- Resistance (Ω)
- Power (W)

## Formulas
- Ohm: `I = V / R`
- Power: `P = V * I`
- Series: `R_total = ΣR`
- Parallel: `1/R_total = Σ(1/R)`
